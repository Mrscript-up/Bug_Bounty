### This tool can cover and automate the process of subdomain collection, testing, liveness, and cleanup.
### This tool is a simple automation tool, but we will try to improve it in the future.

## Prerequisites:

- subfinder
- dnsx
- naabu
- httpx

## run:
```bash
python3 _main_.py example.com
```
## output:
```bash
{domain}_naabu_results.txt => domains_ports
{domain}_httpx_results.txt => online domains
{domain}_httpx_final_results.txt => httpx scan
```

#### then you will have subdomains , ports , soma information from httpx scan.
